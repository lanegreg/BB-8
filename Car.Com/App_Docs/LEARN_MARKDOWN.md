##**Some Sample Markdown:**

#H1
##H2
###H3
Plain text looks like: Mary had a little lamb!  

Inline code:  Call `init(this);` at app start.  

Code looks like this.

        var myMod = (function() {
          var retVal = "Hello World!";
          var init = function() {
            return retVal;
          }

          return {
            init: init
          };
        )();
